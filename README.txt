Manuscript Title: Analyzing differences between restricted mean survival time
curves using pseudo-values 
Authors: Federico Ambrogi, Simona Iacobelli, Per Kragh Andersen 
Code by: Federico ambrogi (federico.ambrogi@unimi.it)

This README file presents the computer code in R for the manuscript "Analyzing
differences between restricted mean survival time curves using pseudo-values".
The code in file "code_reproducible.R" reproduces figures 2 and 3 of the manuscript.

Required R packages: pseudo, splines, multcomp, survival, geepack, survRM2.
